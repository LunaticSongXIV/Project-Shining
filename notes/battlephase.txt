global.battlephase is a variable that sets which phase of the battle engine is currently running.

Controlling this variable is vital, any map with a battle controller will automatically engage > 0

battlephase 0 = no battle is currently in progress
battlephase 1 = determine which character's turn should happen next
battlephase 2 = movement
battlephase 3 = action
battlephase 4 = battle scene
battlephase 5 = evaluation
